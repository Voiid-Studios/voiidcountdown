---
title: "PAUSE"

---

## Usage Example

<CodeGroup>

```java Java Example
@EventHandler
public void onVCTEvent(VCTEvent event) {
    if (event.getType() == VCTEventType.PAUSE) {
        String timerId = event.getTimerId();

        Bukkit.broadcastMessage("The timer " + timerId + " has been paused!");
    }
}
```

```yaml CE Example
vct_pause_example:
  type: custom
  custom_event_data:
    event: voiidstudios.vct.api.VCTEvent
    variables_to_capture:
    - '%eventType%;getType()'
    - '%timerId%;getTimerId()'
  conditions:
  - '%eventType% == PAUSE'
  actions:
    default:
    - 'to_all: actionbar: The timer %timerId% has been paused!;100'
```

</CodeGroup>

## Behavior

* Fired when the active timer is paused (via command or API).
* The countdown freezes until resumed.
