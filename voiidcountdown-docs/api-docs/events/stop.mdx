---
title: "STOP"
description: "Triggered when a timer is stopped manually."
---

## Usage Example

<CodeGroup>

```java Java Example
@EventHandler
public void onVCTEvent(VCTEvent event) {
    if (event.getType() == VCTEventType.STOP) {
        String timerId = event.getTimerId();

        Bukkit.broadcastMessage("The timer " + timerId + " has been stopped!");
    }
}
```

```yaml CE Example
vct_stop_example:
  type: custom
  custom_event_data:
    event: voiidstudios.vct.api.VCTEvent
    variables_to_capture:
    - '%eventType%;getType()'
    - '%timerId%;getTimerId()'
  conditions:
  - '%eventType% == STOP'
  actions:
    default:
    - 'to_all: actionbar: The timer %timerId% has been stopped!;100'
```

</CodeGroup>

## Behavior

* Fired when the active timer is stopped and removed manually.
* Does not trigger `FINISH`, since it is considered an interruption.
